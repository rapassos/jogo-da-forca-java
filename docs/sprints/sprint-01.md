# Sprint 01 — Fundação da evolução para interfaces paralelas

## Objetivo
Estabelecer a base técnica e documental para manter a versão desktop atual e evoluir em paralelo para uma interface web moderna.

## Entregas
- Bootstrap de execução com suporte a modo desktop e web.
- Estrutura inicial para futuras interfaces web.
- Prototipação inicial de uma interface web local com servidor embutido.
- Documentação de roadmap, changelog e contribuição.
- Base de testes para validar a seleção de modo.

## Resultado
A aplicação continua rodando como projeto Java e ganhou um primeiro protótipo web local, preparando a base para a próxima sprint de refinamento visual e experiência de usuário.
