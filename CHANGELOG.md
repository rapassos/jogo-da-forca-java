# Changelog

## [Unreleased]

### Added
- **Deploy em produção:** aplicação rodando no Render em https://jogo-da-forca-java.onrender.com
- Dockerfile multi-stage para containerização
- Suporte a porta configurável via variável de ambiente `PORT`
- Estrutura inicial para suporte a múltiplas interfaces: desktop e web
- Bootstrap de aplicação com seleção de modo via argumento de execução
- Prototipação inicial da interface web com servidor local e página HTML simples
- Suporte inicial a entrada de letras via teclado do computador na interface web
- Interface web com estética de caderno antigo e personagem animado
- Documentação de roadmap e sprints para evolução do projeto

- Atualização de créditos e referências para o Dicionário Aberto
